                                                 ---Anaconda is best for resume reader application---

---Install---

	Install Anaconda --> https://www.anaconda.com/distribution/#download-section


---Install the following python packages---
1.Open Anaconda prompt and install the following packages
	python -m pip install --upgrade pip  or  pip install --upgrade pip
	pip install PyMuPDF
	pip install docx2txt
	pip install Cython
	pip install pyresparser(if error thrown, install that untill successfully installed)
	pip install PyPDF2
	pip install textract(if error thrown, install that untill successfully installed)
	pip install docx
	pip install python-docx 
	pip install pywin32
	python -m spacy download en
	conda install -c conda-forge poppler


---Run---
-->Open Anaconda prompt

	1.change directory to the program directory --> cd /folderpath
	2.Run app.py file in directory
	3.copy the https:// path... link is like http://127.0.0.1:5000/
	4.open browser and paste the link in address bar
	5.Then Resume Reader Application is run....
